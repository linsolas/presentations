Les avantages

  • Cloisonnement du CSS
  • Mécanisme de composition
  • Standard W3C

Les inconvénients

  • Support des navigateurs
  • Polyfills complexes
  • L'encapsulation du CSS par les CSS Modules
Les modules CSS sont une "bidouille", à savoir une modification dynamique du nom des classes pour éviter toute collision. Ce n'est pas un standard.
