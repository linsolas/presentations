Shadow DOM?

Document Object Model

DOM = interface permettant à des scripts d'accéder et de mettre à jour la structure, le contenu et les styles du document.

Shadow DOM ?

Mais pourquoi ?

(1) Isoler le DOM


Depuis l'extérieur, cacher l'arbre des composants du shadow DOM.

(2) Contrôler la portée du CSS


  • Le CSS de la page n'impacte pas le composant.
  • Le CSS du composant n'impacte pas la page.
